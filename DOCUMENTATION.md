## ✨ Key Features

### 1. Cinematic Animations
The site uses **GSAP** to create smooth, high-end entries for text and images, giving it a premium agency feel.

### 2. Custom Magnetic Cursor
A custom circular cursor follows the mouse movement, with a "magnetic" effect on buttons and links to enhance interactivity.

### 3. Stacking Cards (Services)
In the services section, cards stack on top of each other as you scroll, creating a modern and engaging way to present expertise.

### 4. Horizontal Scroll (Projects)
The portfolio section uses a horizontal scrolling layout to showcase projects in a cinematic, wide-screen format.

### 5. Responsive Design
Built with a mobile-first approach using Tailwind's responsive utilities (`md:`, `lg:`, etc.).

## 🛠️ How to Maintain
- **Styles:** Modify `css/style.css` for custom CSS or update the `tailwind.config` in the `<head>` of HTML files for theme colors.
- **Functionality:** All animation logic and interactive elements are handled in `js/script.js`.
- **Images:** Add new images to `assets/images/` and reference them accordingly.
